# WP Education

A complete education management system for WordPress

## Minimum Requirement
 - PHP 5.2.0
 - WordPress 4.0+

## Installation

* Clone the repository inside `/wp-content/plugins/`
* CD into folder `cd wp-erp` and run `composer install`
* Lastly run `composer dump-autoload -o`
