== Description ==

A complete education management system for WordPress

= Key Features =

* Manage student (Add/edit/delete/view profile)
* Manage teacher (Add/edit/delete/view profile)
* Manage parents (Add/edit/delete/view profile)
* Notice management
* Class management
* Subject management
* Routine management
+ Result Management
* Transport manamgent
* Extended Settings Option
* All version of WordPress 4.4+ supported

== Installation ==

= New Install via FTP =

1. Unzip the package wp-school-management.zip on you computer.
2. Upload the unzipped folder the /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress.
4. Goto Settings menu and set your options.

= New Install via the WordPress uploader =

1. Click Plugins > Add New inside of your WordPress install.
2. Click Upload and select the package wp-school-management.zip
3. Activate the plugin through the Plugins menu in WordPress.
4. Goto Settings menu and set your options.

= Updating via the FTP =

1. If you've made any custom changes to the core you'll need to merge those changes into the new package.
2. Unzip the package wp-school-management.zip on you computer.
3. Replace the current folder in your plugins folder with the new unzipped folder from your computer.
